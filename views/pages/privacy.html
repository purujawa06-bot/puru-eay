✦ Privacy Policy

Terakhir diperbarui: Januari 2025

1. Data yang Kami Kumpulkan

Kami mengumpulkan data sesedikit mungkin:

  • Session ID anonim — Cookie terenkripsi untuk keamanan permintaan (tidak terhubung ke identitas asli kamu).
  • Alamat IP — Digunakan sementara untuk rate limiting, tidak disimpan secara permanen.
  • Riwayat chat — Disimpan hanya di browser kamu (localStorage), tidak di server kami.

2. Cara Kami Menggunakan Data

Data yang dikumpulkan digunakan semata-mata untuk:

  • Memproses permintaan chat dan meneruskannya ke model AI.
  • Mencegah penyalahgunaan dan serangan otomatis (rate limiting, token CSRF).

Kami tidak menjual, menyewakan, atau membagikan data kamu kepada pihak ketiga untuk tujuan komersial.

3. Cookie

PurAI menggunakan satu cookie sesi (purai_sid) yang bersifat httpOnly dan ditandatangani secara kriptografis. Cookie ini tidak melacak aktivitas browsing di luar platform.

4. Penyimpanan Lokal

Riwayat percakapan tersimpan di localStorage browser kamu. Kamu memiliki kendali penuh — hapus kapan saja melalui pengaturan browser atau tombol hapus di sidebar.

5. Keamanan

Kami menerapkan langkah-langkah teknis termasuk token CSRF berbasis HMAC-SHA256, header keamanan HTTP, dan pemblokiran agen otomatis untuk melindungi layanan.

6. Hak Kamu

Karena kami tidak mengaitkan data dengan identitas, tidak ada profil pengguna yang perlu dihapus. Kamu bisa menghapus data lokal kapan saja langsung dari browser.

7. Perubahan Kebijakan

Kebijakan ini dapat diperbarui. Perubahan signifikan akan dikomunikasikan melalui platform sebelum berlaku.
